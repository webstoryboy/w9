Box-shadow

box-shadow : [h-shadow] [v-shadow] [blur] [spread] [color] [inset] / none;
속성 설명
h-shadow 그림자의 수평(X)축 거리를 나타냅니다.
v-shadow 그림자의 수직(Y)축 거리를 나타냅니다.
blur 그림자의 흐림 정도를 나타냅니다.
spread 그림자의 거리를 나타냅니다.
color 그림자의 색을 나타냅니다.
inset 그림자의 효과를 내부로 나타냅니다.
