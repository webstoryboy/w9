Blend Tool(W)

  • Smooth color : 부드러운 색상 : 블렌드에 대한 단꼐 수가 자동으로 설정
  • Specified Steps : 지정된 단계 : 블렌드의 시작과 끝 사이의 단계 수를 설정
  • Spefied Distance : 지정된 거리 : 블렌드의 시작과 끝 단계 사이의 거리를 조절
  • Orientation : 블렌드의 오브젝트 방향을 설정
    • 페이지에 정렬 : 블렌드 방향을 페이지의 X축에 수직으로 설정
    • 패스에 정렬 : 패스에 수직으로 블렌드의 방향을 설정
  • [object] - [Blend]
    • [Revers Spine] : 블렌드 흐름의 방향을 반전
    • [Revers Front to Back] : 블렌드 흐름의 방향은 반전 시키지 않고 앞뒤 순서만 바꿈

참고



라이브 페인트 이용하여 색상채우기

  1. 그림을 그린 후 [Object] - [Live Paint] - [Make]를 선택합니다.
  2. Live Paint Bucket(K) 툴을 선택하여 색을 채웁니다.
  3. 색상을 다 채우면 [Object] - [Live Paint] - [expand]를 선택하여 해제합니다.


swatches를 이용한 다양한 색상채우기

  1. [window] - [swatches] - swatches libraries menu 아이콘을 클릭합니다.
  2. 색상 스타일에 맞는 스와치를 골라 색을 표현합니다.
  3. [window] - [swatches] - Open Color Themes Panel 아이콘을 클릭합니다.
  4. 유사한 색상을 선택하여 색을 표현합니다.
  5. 대상을 선택한 후 recolor artwork를 선택한 후 색상을 변경합니다.


황금비율을 이용한 로고 만들기

  1. 1.168의 비율을 이용하여 사각형과 원을 그린다.
  2. 원만 모아서 원하는 로고 모양을 그린다.
  3. pathfinder를 이용하여 필요없는 부분을 제거한다
  4. unite를 이용하여 면을 만들고 색을 넣어준다.
  5. gradient를 이용하여 색상을 넣어준다.
