이벤트(Event)

  • .click() : 클릭해보세요!
  • .dblclick() : 더블 클릭해보세요!
  • .hover() : 마우스 오버해보세요!
  • .mousedown() : 마우스 버튼을 놀러보세요!
  • .mouseenter() : 마우스를 여기에 올려보세요!
  • .mouseleave() : 마우스를 벗어나보세요!
  • .mousemove() : 마우스를 움직여보세요!
  • .mouseout() : 마우스를 벗어나보세요!
  • .mouseover() : 마우스를 올려보세요!
  • .mouseup() : 마우스를 눌렀다가 떼어보세요!
