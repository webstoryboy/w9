SASS

Sass는 세계에서 가장 성숙하고 안정적이며 강력한 전문적인 CSS 확장 언어입니다.

설치하기

  • http://rubyinstaller.org/에 방문해서 ruby를 설치합니다.
  • 윈도우키+R키를 누르고 cmd를 입력합니다.
  • gem install sass 를 입력해서 sass 컴파일러를 설치합니다.

맥 설치하기

  • 터미널에서 gem install sass 를 입력해서 sass 컴파일러를 설치합니다.
  • 권한이 없다고 할 경우 sudo gem install sass 를 입력합니다.
