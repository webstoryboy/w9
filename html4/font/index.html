Font

폰트 종류

  • TrueTypeFonts(TTF): 트루타입은 애플과 마이크로소프트에 의해, 1980년대 후반에 개발된 글꼴 표준입니다.
  • OpenTypeFonts(OTF): 오픈타입은 확장가능한 마이크로소프트 컴퓨터 글꼴 형식입니다.
  • The Web Open Font Format(WOFF): WOFF는 W3C에서 추천하는 글꼴 형식입니다.
  • The Web Open Font Format(WOFF2.0): WOFF2는 WOFF에서 향상된 글꼴입니다.
  • SVG Fonts/Shapes: SVG는 벡터 방식의 글꼴형식입니다.
  • Embeded Open Type Fonts(EOT): 익스플로러 6~8에서 사용하던 웹 푠트 방식입니다.
