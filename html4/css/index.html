CSS

CSS

CSS(Cadcading Style Sheet)은 HTML을 꾸며주는 스타일시트입니다.

  • HTML이 웹 사이트의 몸체를 담당한다면, CSS는 옷과 액세사리 같은 꾸미는 역할을 담당합니다.
  • HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 사이트처럼 꾸밀 수 있습니다. csszengarden
  • CSS3는 그림자 효과, 그라데이현, 변셩 등의 효과를 사용할 수 있으며, 애니메이션 기능도 사용 할 수 있습니다.

CSS Selector

컨텐츠(요소)를 선택하는 방법을 정의합니다.

  • id 선택자
    • 하나의 태그만 식별하기 위한 선택자입니다.
    • id는 한 요소에 하나만 사용할 수 있습니다.
    • CSS에서는 "#"으로 표시합니다.
  • class 선택자
    • 여러개의 태그를 중복해서 사용하는 선택자입니다.
    • class 한 요소에 두개 이상 사용할 수 있습니다.
    • CSS에서는 "."으로 표시합니다.
  • * 선택자

CSS Syntax

div {color:red;}

  • CSS 크게 선택자(Selector), 속성(property), 값(Value)으로 구성되어 있습니다.
  • 선택자(Selector) : 어떤 요소를 선택하기 위한 방법입니다.
  • 속성(property) : 선택한 요소한테 속성을 줄 수 있습니다.
  • 값(Value) : 선택한 요소에 대한 값을 설정합니다.

float으로 인한 영역깨짐(height:0) 방지법

  • 깨지는 영역에 똑같이 float을 사용합니다(X).
  • float의 성질을 차단하는 clear:both를 사용합니다(X).
  • float을 사용한 부모박스한테 overflow:hidden을 사용합니다.
  • float을 사용한 부모박스한테 clearfix를 설정합니다.

컨텐츠 요소를 안보이게 하는 방법

  1. 1. display: none; <-> display : block; | 영역X | 애니X
  2. 2. visibility: hidden; <-> visibility: visible; | 영역O | 애니X
  3. 3. opacity: 0; <-> opacity: 1; | 영역O | 애니O
  4. 4. ir(width:0; height:0)

float 속성은 블록요소의 정렬 상태를 설정합니다.

  • float: none; float의 성질을 적용하지 않습니다.
  • float: left; 블록요소를 왼쪽으로 정렬합니다.
  • float: right; 블록 요소를 오른쪽을 정렬합니다.

margin

margin 속성의 요소의 바같쪽 여백을 설정합니다.

  • margin: length; 요소의 바같쪽 여백을 CSS 단위를 통해 설정합니다.
  • margin: %; 요소의 바같쪽 여백을 %를 통해 설정합니다.
  • margin: auto; 요소의 바같쪽 여백을 자동으로 설정합니다.
  • margin-top: 10px;
  • margin-right: 11px;
  • margin-bottom: 12px;
  • margin-left: 13px;
  • margin : 10px 11px 12px 13px; (위, 오른쪽, 아래, 왼쪽)
  • margin : 10px 11px 12px; (위10, 왼쪽오른쪽11, 아래12)
  • margin : 10px 11px; (위,아래10, 왼쪽오른쪽11)
  • margin : 10px; (위,오른쪽,아래,왼쪽:10)
  • margin : 0 auto (블록구조에서 가운데 정렬)

background

background 속성은 배경 속성 값들을 설정합니다.


background-attachment

background-attachment 속성은 배경 이미지의 고정여부를 설정합니다.

  • background-attachment : scroll(기본값)
  • background-attachment : fixed : 배경이미지만 고정
  • background-attachment : local : 배경이미지와 콘텐츠 고정

background-blend-mode

background-blend-mode 속성은 배경을 혼합했을때의 상태를 설정합니다.

  • background-blend-mode : normal(기본값)
  • background-blend-mode : multiply
  • background-blend-mode : screen
  • background-blend-mode : overlay
  • background-blend-mode : darken
  • background-blend-mode : lighten
  • background-blend-mode : color-dodge
  • background-blend-mode : saturation
  • background-blend-mode : color
  • background-blend-mode : luminosity

background-clip

background-clip 속성은 백그라운드 색의 영역을 설정합니다.

  • background-clip : border-box(기본값); 백그라운드 속성이 테두리 영역까지 포함하여 표현
  • background-clip : padding-box; 백그라운드 속성이 테두리 영역을 제외하여 표현합니다.
  • background-clip : content-box; 백그라운드 속성이 내부 콘텐츠 영역까지 표현합니다.

background-color

background-color 속성은 백그라운드 색을 설정합니다.


background-image

background-image 속성은 백그라운드 이미지 속성을 설정합니다.

  • background-image : url("주소")
  • background-image : none(기본값)

background-origin

background-origin 속성은 백그라운드 이미지 영역 속성을 설정합니다.

  • background-origin : border-box(기본값); 배경 이미지의 위치를 테두리 영역에 포함하여 설정합니다.
  • background-origin : padding-box; 배경 이미지의 위치를 패딩 영역에 위치하여 설정합니다.
  • background-origin : content-box; 배경 이미지의 위치를 컨텐츠 영여게 위치하게 설정합니다.

background-origin은 백그라운드 이미지의 영역 속성을 설정하는 방면, background-clip은 백그라운드 색 속성을 설정하는 방법이 유사합니다.


background-position

background-position 속성은 백그라운드 위치 속성을 설정합니다.

  • background-position : left top
  • background-position : top right
  • background-position : center center
  • background-position : 10% 50%
  • background-position : 0px 10px
  • background-position : 0px 10px, 20px 10px

background-repeat

background-repeat 속성은 백그라운드 이미지 반복여부를 설정합니다.

  • background-repeat : repeat(기본값) : 이미지를 x축과 y축으로 반복 설정
  • background-repeat : repeat-x : 이미지를 x축으로 반복 설정
  • background-repeat : repeat-y : 이미지를 y축으로 반복 설정
  • background-repeat : no-repeat : 이미지를 반복하지 않습니다. 이미지가 한번만 나옵니다.

background-size

background-size 속성은 백그라운드 이미지 사이즈를 설정합니다.

  • background-size : auto(기본값)
  • background-size : 120px(length)
  • background-size : 80%(percentage)
  • background-size : cover : 이미지를 요소 크기에 맞게 공백없이 설정됩니다.
  • background-size : contain : 이미지를 요소 크기에 맞게 가로 세로 비율을 유지하면서 설정됩니다.
