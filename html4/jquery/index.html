제이쿼리

제이쿼리는 자바스크립트 라이브러리를 이용해 만든 언어입니다. 라이브러리는 자주 사용하는 코드를 재활용 및 효율성 있게 사용할 수 있도록 만들어진 다양한 함수의 집합을 의미합니다.

제이쿼리는 HTML에 포함되어 있는 클라이언트 사이드 스크립트 언어를 단순화하도록 설계된, 웹 브라우저 호환성을 가진 자바스크립트 라이브러리입니다.

제이쿼리의 장점

  1. 제이쿼리는 CSS를 쉽게 적용할 수 있습니다. CSS를 사용할 줄 안다면 제이쿼리에서도 사용하는 방법이 동일하기 때문에 쉽게 적용할 수 있습니다.
  2. 제이쿼리는 크로스브라우징을 지원합니다. 자바스크립트는 익스플로러 버전마다 코드가 틀리지만 제이쿼리는 구버전의 익스플로러도 지원됩니다.
  3. 제이쿼리는 플러그인이 풍부합니다. 제이쿼리는 자바스크립트 라이브러리 중 가장 많이 사용하며, 가장 많은 플러그인을 가지고 있습니다.
  4. 제이쿼리는 코드를 적게, 효율적으로 작성할 수 있습니다. 제이쿼리는 자바스크립트에 비해 소스슬 더 적게 작성하여도 똑같은 효과를 구현 할 수 있습니다.
  5. 제이쿼리는 좋은 확장성과 ajax기능을 구현합니다.

제이쿼리 사용방법

제이쿼리를 사용하기 위해서는 자바스크립트 라이브리러 파일이 필요합니다.

라이브러리 충돌

$(엘리먼트)와 jQuery(엘리먼트)는 같은 의미이미만 $를 사용하는 다른 라이브러리들과의 충돌 때문에 다음과 같은 방법을 사용합니다.

$.noConflict();를 사용합니다. 이 메서드는 jQuery객체와 $기호의 연결을 끊어서 $를 쓰는 다른 라이브러리와 충돌하지 않게 합니다.

선택자

제이쿼리는 선택자를 HTML을 통해 선택하여, 제이쿼리 기능을 쉽게 구현 할 수 있다.

$("선택자").메서드(매개변수,함수);

기본 선택자

종류 예시 설명
* $("*") 전체 태그를 선택합니다.
태그 선택자 $("p") 기본 태그를 선택합니다.
클래스 선택자 $(".class") 클래스 태그를 선택합니다.
클래스 선택자 $("#id") 아이디 태그를 선택합니다.
그룹 선택자 $("#id, .class") 여러가지 태그를 선택합니다.

계층 선택자

종류 예시 설명
자손 선택자 $("div ul li") 자식(하위) 태그를 모두 선택합니다.
child 선택자 $("div > ul") 자식(하위) 태그만 선택합니다.(자손 포함안됨)
sibling 선택자 $("div + p") 자식 태그 다음 형제 태그를 선택합니다.
siblings 선택자 $("div ~ p") 자식 태그 다음 모든 형제 태그를 선택합니다.

속성 선택자

종류 예시 설명
[name='value'] $("li a[href='#page']") 속성 중에 #page와 일치하는 태그를 선택합니다.
[name^='value'] $("li a[href^='http']") 속성 중에 http로 시작하는 태그를 선택합니다.
[name$='value'] $("li a[href$='.com']") 속성 중에 .com 끝나는 태그를 선택합니다.
[name*='value'] $("li a[href*='web']") 속성 중에 'web'이 포함되어 있는 태그를 선택합니다.
[name!='value'] $("li a[href!='naver.com']") 속성 중에 'naver.com'이 포함되어 있는 않는 태그를 선택합니다.
[name='value'][name='value'] $("li a[href][class]) 속성 중에 'href'속성과 'class' 속성이 있는 태그를 선택합니다.

기본필터 선택자

종류 설명
:animated show, hide, slideDown 등의 애니메이션 태그를 선택
:eq(index) 선택된 태그들의 인덱스 번호를 통해 선택
:gt(index) 선택된 집합에서 인덱스 번호보다 큰 인덱스를 가지고 있는 태그 선택
:lt(index) 선택된 집합에서 인덱스 번호보다 작은 인덱스를 가지고 있는 태그 선택
:header 제목 요소(h1~h6) 태그를 선택
:first 선택된 요소 중에서 첫 번째 요소를 선택
:last 선택된 요소 중에서 마지막 번째 요소를 선택
:odd 선택된 요소 중에서 홀수 번째 요소를 찯아 선태
:even 선택된 요소 중에서 짝수 번째 요소를 찯아 선택
:not 현재 선택한 요소의 반대 요소를 선택

내용필터 선택자

종류 설명
:contains() ()안의 텍스트와 일치하는 문자열의 요소가 있을 때 선택
:empty 요소에 텍스트가 없을 때 선택
:has() 요소 내부에 찾고 싶은 태그를 후손 요소까지 살펴본 후 있으면 선택
:parent 요소에 텍스트가 있을 때 선택

보임필터 선택자

컨텐츠 영역을 보이지 않게 방법

  • display: none : 영역X : 모션X
  • visibility: hidden; 영역O : 모션X
  • opacity: 0; 영역O : 모션O
  • width:0; height: 0; 영역X
  • form 요소 중에 type="hidden"
  • 부모요소가 보이지 않거나 숨겨져 있을 때

visibility: hidden;, opacity: 0;은 위치가 제거되지 않기 때문에 :hidden 선택자에게 선택되지 않습니다.

종류 설명
:hidden 보이지 않는 요소를 선택
:visible 보이는 요소를 선택

자식필터 선택자

종류 설명
:first-child 첫 번째 자식 요소를 선택
:last-child 마지막 번째 자식 요소를 선택
:nth-child(index) index 번째 자식 요소를 선택
:nth-child(even) 짝수 번째 자식 요소를 선택
:nth-child(odd) 홀수 번째 자식 요소를 선택
:nth-child(2n+3) 2n+3 번째 자식 요소를 선택
:only-child 자식 요소가 오직 하나인 요소를 선택

폼필터 선택자

종류 설명

탐색

제이쿼리의 선택자를 기준으로 선택한 요소 중 원하는 요소를 다시 한번 더 선택할 수 있는 탐색과 관련된 제이쿼리 메서드입니다. 탐색은 크게 트리구조와 필터로 나눌 수 있습니다.

트리구조 탐색(Tree Traversal)

Tree 구조를 기반으로 찾아가는 방식입니다.

종류 설명
.childred() 선택한 요소의 모든 자식을 선택합니다.(자손 포함안됨)
.find() 선택한 요소의 자손 요소 중 조건에 맞는 요소를 선택합니다.
.next() 선택한 요소의 다음 요소를 선택합니다.
.nextAll() 선택한 요소의 다음 모든 요소를 선택합니다.
.parent() 선택한 요소의 부모 요소를 선택합니다.
.parents() 선택한 요소의 모든 부모 요소를 선택합니다.
.prev() 선택한 요소의 이전 요소를 선택합니다.
.prevAll() 선택한 요소의 이전 요소를 선택합니다.
.closest() 선택한 요소를 포함하면서 가장 가까운 상위 요소를 선택합니다.
.nextUntil() 다음에 위치한 요소를 조건에 맞을 때 찾습니다.
.parentsUntil() 조건이 참이 될 때까지 부모 요소를 찾습니다.
.prevUntil() 이전에 위치한 요소를 조건에 맞을 때 찾습니다.
.siblings() 형제 요소를 모두 찾습니다.

필터링(Filtering)

종류 설명
.eq() 선택한 요소의 인덱스 번호를 갖는 요소를 선택합니다.
.filter() 선택한 요소에서 한번더 필터링하여 요소를 선택합니다.
.has() 선택한 요소의 자식 요소에서 주어진 선택자를 찾습니다.
.is() 선택한 요소의 상태가 지정한 속성과 일치하면 선택합니다.
.last() 선택한 요소의 마지막 요소를 선택합니다.
.map() 배열의 데이터를 변환시켜 새로운 배열 객체로 만듭니다.
.not() 선택한 요소만 제외하고 나머지를 선택합니다.
.slice() 선택한 요소만 지정해서 선택합니다.

기타 탐색(Miscellaneous Traversing)

종류 설명
.add() 선택한 요소에 새로운 요소를 추가로 선택합니다.
.addBack() 선택한 요소의 상위 요소도 추가로 선택합니다.
.contents() 선택한 요소의 하위 요소도 추가로 선택합니다.
.each() 선택한 여러 요소들을 각각 순차적을 선택합니다.
.end() 선택한 요소의 필터링이 일러나기 이전에 요소를 선택합니다.

.after() 선택한 요소 다음 위치에 새로운 요소를 추가합니다.

.remove 선택한 요소 삭제합니다.
.detach 선택한 요소 삭제합니다.
.unwrap 선택한 요소의 부모박스를 삭제합니다.
.replaceAll 선택한 요소를 새로운 요소로 바꿉니다.
.replaceWith 선택한 요소를 새로운 요소로 바꿉니다.
.html() 선택한 요소 내부의 HTML을 읽고 쓸 수 있습니다.
.text() 선택한 요소의 텍스트를 읽고 쓸 수 있습니다.

ATTRIBUTES

문서 객체의 속성과 관련된 기능을 처리하는 메서드입니다.

종류 설명
Attributes .attr() 새로운 속성을 생성하거나 기존의 속성을 변경할 때 사용합니다.
.prop() 선택한 요소에 속성을 반환하거나, 생성, 변환할 때 사용합니다.
.removeAttr() 선택한 요소에서 기존의 속성을 삭제합니다.
.removeProp() 선택한 요소에서 기존의 속성을 삭제합니다.
.val() 입력 요소에 있는 속성 값을 불러오거나 변경할 때 사용합니다.
Demension .width() 선택한 요소의 가로 크기를 반환하거나 변경합니다.
.height() 선택한 요소의 세로 크기를 반환하거나 변경합니다.
.innerWidth() 선택한 요소의 패딩값을 포함한 가로 크기를 반환하거나 변경합니다.
.innerHeight() 선택한 요소의 패딩값을 포함한 세로 크기를 반환하거나 변경합니다.
.outerWidth() 선택한 요소의 패딩값 + 보더값을 포함한 가로 크기를 반환하거나 변경합니다.
.outerHeight() 선택한 요소의 패딩값 + 보더값을 포함한 세로 크기를 반환하거나 변경합니다.
Offset .offset() 웹 문서를 기준으로 위치값을 읽어오며, left와 top의 속성을 통해 x축과 y축의 좌표를 알려줍니다.
.position() 기준점을 통해서 선택한 요소의 위치값을 읽어오며, left와 top의 속성을 통해 x축과 y축의 좌표를 알려줍니다.
.scrollLeft() 브라우저 요소의 가로 스크를 이동 값을 가져옵니다.
.scrollTop() 브라우저 요소의 세로 스크를 이동 값을 가져옵니다.
