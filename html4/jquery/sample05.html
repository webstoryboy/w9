Select

  • 전체 선택자(*)
  • child 선택자 : 자식(하위)태그만 선택한다. (자손 포함 안됨)
  • sibling 선택자 : 자식(하위)태그 다음 형제 태그를 선택한다.
  • siblings 선택자 : 자식(하위)태그 다음 모든 형제 태그를 선택한다.
  • [name='value'] : 속성 중에 #page와 일치하는 태그를 선택한다.
  • [name^='value'] : 속성 중에 http로 시작하는 태그를 선택한다.
  • [name$='value'] : 속성 중에 .com으로 끝나는 태그를 선택한다.
  • [name*='value'] : 속성 중에 'web'이 포함되어있는 태그를 선택한다.
  • [name!='value'] : 속성 중에 'naver.com'이 포함되어있지 않은 태그를 선택한다.
  • [name!='value'][name!='value'] : [name!='value'][name!='value']
  • :animated : show,hide,slideDown 등의 애니메이션 태그를 선택
  • :eq(index) : 선택된 태그들의 인덱스 번호를 통해 선택
  • :gt(index) : 선택된 집합에서 인덱스 번호보다 큰 인덱스를 가지고 있는 태그 선택
  • :lt(index) : 선택된 집합에서 인덱스 번호보다 작은 인덱스를 가지고 있는 태그 선택
  • :header : 제목 요소(h1~h6) 태그를 선택
  • :first : 선택된 요소 중에서 첫 번째 요소를 선택
  • :last : 선택된 요소 중에서 마지막 요소를 선택
  • :odd : 선택된 요소 중에서 홀수 번째 요소를 선택
  • :even : 선택된 요소 중에서 짝수 번째 요소를 선택
  • :not : 현재 선택한 요소의 반대 요소를 선택
