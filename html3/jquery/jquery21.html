클릭하면 리스트의 높이(.height()) 값을 구합니다. 클릭하면 리스트의 높이(.css("height")) 값을 구합니다. 클릭하면 문서의 높이 값을 구합니다. 클릭하면 브라우저의 높이 값을 구합니다.
클릭하면 리스트의 가로(.width()) 값을 구합니다. 클릭하면 리스트의 가로(.css("width")) 값을 구합니다. 클릭하면 문서의 가로 값을 구합니다. 클릭하면 브라우저의 가로 값을 구합니다.
  • .children() : 선택한 요소의 바로 아래 자식 요소만 선택합니다.
  • .closest() : 선택한 요소를 포함하면서 가장 가까운 상위 요소를 선택합니다.
  • .find() : 선택한 요소의 자식 요소 중 조건에 맞는 요소를 선택합니다.
  • .next() : 선택한 요소를 다음 형제 요소를 선택합니다.
  • .is() : 선택한 요소 객체의 특징을 판별합니다.
  • .last() : 선택한 요소에서 마지막 번째 자식을 선택합니다
  • .filter() : 선택한 요소에서 선택하거나 함수를 사용할 수 있습니다.
  • .first() : 선택한 요소에서 첫 번째 자식을 선택합니다.
  • .has() : 선택한 요소의 자식 요소에서 주어진 선택자를 찾습니다.

.width() .innerWidth() .outerWidth()
